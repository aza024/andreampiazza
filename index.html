Fileshare

Previous Next

About: FileShare is a full-stack file-sharing application built using Node/Express and JavaScript that allows users to upload, download, and share files. Hosted on AWS using EC2, FileShare stores user files using an S3 Bucket with password storage backed by PostgreSQL in RDS. FileShare allows users to safely store their files using JWT tokens for authentications, while also allowing users to securely share their files even with users that do not have accounts.

Features: FileShare has many other features such as the ability to view stored files in user browsers - view images or play mp4 files. User can share secure UUID links with non-users where they can view and download the shared file. FileShare gives users the option to create a .txt file directly in the browser which they can then upload or download to their local file system. Multiple forms of searching and sorting are supported,allowing users to find their files very quickly.

Skills Used: HTML, CSS, Javascript, JQuery, PostgreSQL, SQL, AWS EC2 and S3 Buckets, JWT Tokens, Bcrypt, UUID's
