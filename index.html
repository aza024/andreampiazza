Fileshare

About: FileShare is a full-stack file-sharing application built using Node/Express and JavaScript that allows users to upload, download, and share files. Hosted on AWS using EC2, FileShare stores user files using an S3 Bucket with password storage backed by PostgreSQL in RDS. FileShare allows users to safely store their files using JWT tokens for authentications, while also allowing users to securely share their files even with users that do not have accounts.

Features: FileShare has many other features such as the ability to view stored files in user browsers - view images or play mp4 files. User can share secure UUID links with non-users where they can view and download the shared file. FileShare gives users the option to create a .txt file directly in the browser which they can then upload or download to their local file system. Multiple forms of searching and sorting are supported,allowing users to find their files very quickly.

Skills Used: HTML, CSS, Javascript, JQuery, PostgreSQL, SQL, AWS EC2 and S3 Buckets, JWT Tokens, Bcrypt, UUID's

PetPals

Previous Next

About: PetPals is a social media oriented, full-stack web application that connects pet owners by means of sharing images of their pets. Using Django, Python, Javascript, jQuery, HTML, and CSS, our goal is to create an application that allows users to sign-up, create a profile for their pet, post pictures of their pets, and interact with other users through likes and comments. Users will be able to add friends and explore random photos of pets.

Features: Users are able to create acounts for their Pets using Django forms that ensure proper input, and backend validation of forms. Pet parents are able to edit and customize thier pet's profile. users can upload custom pictures of their pets and view their posts showing captions as the user hovers over each photo. Users can interact with other users by following other pet profiles. The 'feed' page shows all posts from user that the account is following. Users explore the Pet Pals community on the 'explore' page which will show randomly selected photos from non-followed accounts.

Skills Used: HTML, CSS, SASS, Python, Django, PostgreSQL, JavaScript, JQuery, AJAX

Spontaneous

Previous Next

About: Spontaneous is a responsive full-stack application that allows users to sign up for an account and generates three random Meetups from a profile completed at sign up. Spontaneous queries the Meetup API and returns a list of top 100 Meetups for each interest. From interests select at signup, Spontaneous returns three random Meetups based on the specified interests.

Features: User can create accounts that are validated for accuracy and completion. Users can specify the Meetups that occur within a specific location. Spontaneous will generate and display three random meetups to users based on individually customized profile. Users can add a randomly generated Meetup to their 'saved' meetup section and remove a meetup from the 'saved' section. If the user still can't decide, they can have the app randomly select one of the three randomly generated Meetups.

Skills Used: HTML, CSS, SASS, Javascript, AJAX, Mongo, Mongoose, Node, GeoCoding from Google Maps API, API Key Signatures, Meetup.com's API
